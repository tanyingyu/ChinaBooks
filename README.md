## 项目起源
  [ChinaTextbook](https://github.com/TapXWorld/ChinaTextbook)

## 项目的由来
虽然国内教育网站已提供免费资源，但大多数普通人获取信息的途径依然受限。本来就是公共资源，却有无耻之人，利用这些公共资源，在某些站上积分兑换，变向会员绑定，直接**销售**、直接牟利、变向牟利、私加**水印**等可耻行为。为了应对这种情况，我计划将这些资源集中并整理，以促进义务教育的普及和消除地区间的教育差异。
